In this project we aim to build a website about movies. Users can get information about the top 250 movies, movies that are currently in cinemas, statistics about movies that have been added to the personal watched account and much more. To get the required information, we access several APIs, that we want to give credit in the following part.

IMDb API

We used to IMDb API for the following information:

  • Top 250 movies
  • In Theaters
  • YouTube Trailer
  • Wikipedia

The Movie Database (TMDB)

This product uses the TMDB API but is not endorsed or certified by TMDB.
